
# FARMIA - Smart Agriculture Platform (Desktop Version)

## Description

FARMIA Desktop is a smart agriculture management application developed using Java and JavaFX to provide farmers and administrators with a modern and efficient management system.

The application integrates AI-powered features to simplify farm operations and improve agricultural productivity.

## Technologies Used

* Java
* JavaFX
* MySQL
* REST APIs
* Python
* OpenCV
* AI Integrations

## Main Features

* Secure Authentication
* Facial Recognition Login
* User Management
* Dashboard & Statistics
* Audit & Logs System
* Smart Notifications
* Farm Monitoring
* AI Chatbot Assistance
* Export PDF & Excel
* Role-Based Access Control

## Installation

### Clone the Repository

```bash
git clone <repository-link>
```

### Run the Application

```bash
mvn clean install
```

```bash
mvn javafx:run
```
